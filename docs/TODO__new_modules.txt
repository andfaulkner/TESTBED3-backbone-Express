 pgcli
-- makes it easier to play with postgres
    -- SQL autocompletes!
    -- syntax highlighting!
- repo: https://github.com/dbcli/pgcli
- site: http://pgcli.com/
-- to install:
    brew install pgcli
...then...
    pgcli postgres
    pgcli isight
    pgcli isight_audit
    pgcli template1
